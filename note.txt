-nom String

- addresse : String

Journee
-jour: int (1)
-mois: int (1)
-annee: int (1)

DPS
-id: long (1)
-horaire_depart: int[2]
-horaire_fin: int[2]

Site
-code: string (1)
-nom: string
-longitude: float
-lattitude: float

Sport
-code: string (1)
-nom: string

Competence :
- intitule: string (1)

EstDisponible (Secouriste 0..* - Journee  0..*)
Possede (Secouriste  0..*  Competence  0..*)
EstAffecte (Secouriste  0..* - Competence  0..* - DPS  0..*)
Necessite (Competence  0..* - Competence  0..*)

ALieuDans (DPS  0..* - 1 Site)
